# Prodesk IT Landing Page

A modern, responsive landing page developed as part of **Sprint 1** for the **Prodesk IT Internship Program**. The project showcases a professional IT company website built using **React**, **Vite**, and **Tailwind CSS**, with a focus on responsive design, accessibility, and performance.

## Features

- Fully Responsive Design (Mobile, Tablet & Desktop)
- Sticky Navigation Bar
- Mobile Hamburger Menu
- Hero Section with Call-to-Action
- About Section
- Services Section
- Contact Section
- Footer with Social Media Links
- Dark / Light Theme Toggle
- Glassmorphism Navbar
- Smooth Scrolling Navigation
- Tailwind CSS Styling
- Optimized Images
- Lighthouse Performance & Accessibility Improvements

## Installation

Clone the repository:

```bash
git clone https://github.com/ankitkumarchoudhary01/Prodesk_IT.git
```

Move into the project directory:

```bash
cd Prodesk_IT
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
